# Modelo de Regresión - Diabetes

## Descripción
Este proyecto utiliza un modelo de regresión lineal para predecir la progresión de la diabetes usando el dataset de scikit-learn.

## Variables usadas
- age: Edad (normalizada)
- sex: Sexo (normalizado)
- bmi: Índice de masa corporal
- bp: Presión arterial
- s1-s6: Variables relacionadas con colesterol y glucosa

## Resultados

- MSE: 2900.19
- R2: 0.45
